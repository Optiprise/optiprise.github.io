DS-P01: Future proof ()
The Trust Framework for cross-Domain Data Sharing should be future proof and therefore extensible and non-static.
RationaleA future proof design entails a Trust Framework which supports different implementations and is, to some extent, able to cater for changes in technology, user behaviour, regulation and for a growing number of Data Service Transactions. An adaptive, extensible, and non-static design enables scalability of the Trust Framework.
